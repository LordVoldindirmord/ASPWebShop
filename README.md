ASP Web Shop — Интернет-магазин

- Backend: ASP.NET Core MVC (.NET 10)
- База данных: PostgreSQL + Entity Framework Core (Database First)
- Аутентификация: Cookie Authentication
- Фронтенд: Razor Views, HTML, CSS, JavaScript
- Архитектура: Domain -> DAL -> Service -> Web

Назначения слоев
Слой Domain, назначение: Модели БД, ViewModel, Enum, Response
Слой DAL: DbContext, репозитории, доступ к данным
Слой Service: Бизнес-логика, сервисы
Слой Web: Контроллеры, представления, wwwroot

Роль Customer: Просмотр каталога, корзина, заказы
Роль Seller: Управление своими товарами
Роль Administrator: Управление пользователями и категориями

Функционал:
- Регистрация и авторизация (Cookie)
- Каталог товаров с поиском
- Детальная страница товара (галерея изображений)
- Корзина (добавление, изменение, удаление)
- Оформление заказа + эмуляция оплаты картой
- История заказов
- Панель продавца (CRUD товаров)
- Админ-панель (пользователи и категории)
- Загрузка изображений на сервер (основного, которое будет в каталоге, и дополнительного, которое можно увидеть, перейдя на вкладку "подробнее")

Создание базы данных
Создайте базу данных WebShop в PostgreSQL
Выполните скрипт из файла WebShop.DAL/SqlScripts/init.sql
Укажите строку подключения в appsettings.json

По началу база данных пустая, нужно создать пользователя и с помощью СУБД выдать ему права администратора (сущность User, Role = Administrator), после этого появится панель админа и он сможет выдавать другим пользователям другие роли.
